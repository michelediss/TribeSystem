# Alternatives

This document tracks alternative design paths considered during the project.

## Hardware Alternatives

- Different Raspberry Pi models
- External recorder instead of TribeLog
- Different mixer options
- Different audio interfaces
- External projection mapping tools

## Software Alternatives

- p5.js instead of openFrameworks
- MIDI instead of OSC for internal communication
- Separate FX module instead of integrated FX
- External DAW-based workflow

## Notes

Alternatives are not rejected by default. They remain documented for future comparison.
