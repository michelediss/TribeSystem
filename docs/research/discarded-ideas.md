# Discarded Ideas

This document tracks ideas that were considered and then removed from the current direction.

## Currently Discarded or Deferred

### Separate TribeFX Module

A dedicated external FX module was considered, then merged into TribeBox as an internal FX engine.

### MIDI as Internal Protocol

MIDI was considered for internal communication, then replaced by OSC for TribeBox to TribeVision communication.

### Stutter as v1 Core Feature

Audio-buffer stutter was considered, then deferred. Retrigger covers the main live use case for the first version.

### Fully Laptop-Based Workflow

A laptop-centered workflow was avoided to keep the system focused on standalone hardware.

## Notes

Discarded ideas can be reconsidered later if the project requirements change.
