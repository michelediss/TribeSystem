# Audio Routing

This document describes the default audio routing for Tribe System.

## Default TribeBox Outputs

- Out 1: Kick
- Out 2: Bass / Rumble
- Out 3: Drums
- Out 4: Synth / FX

## Mixer Inputs

- Mackie Ch 1: Kick
- Mackie Ch 2: Bass / Rumble
- Mackie Ch 3: Drums
- Mackie Ch 4: Synth / FX

## Mixer Outputs

The Mackie main outputs go to the PA system.

Additional mixer outputs may be used by TribeLog.
