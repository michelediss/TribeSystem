# Tribe System Overview

Tribe System is a modular hardware and software project for live electronic performance.

## Modules

- TribeBox: audio engine and sequencer.
- TribeVision: visual engine.
- TribeLog: session recorder.
- Mackie Mix12FX: analog mixer.

## Goals

- Fast live setup.
- Portable case-based design.
- Open documentation.
- Modular hardware.
- Clear separation between audio, visuals and recording.
