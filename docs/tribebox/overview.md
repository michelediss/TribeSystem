# TribeBox Overview

TribeBox is the main audio instrument of Tribe System.

## Main Functions

- Sequencer
- Sampler
- Synth engine
- Internal FX engine
- OSC output

## Track Concept

- Track 1: Kick
- Track 2: Bass / Rumble
- Track 3-6: Drums and Percussion
- Track 7: Acid / Lead
- Track 8: Utility / Samples

## Planned Features

- Step sequencing
- Euclidean sequencing
- Generative pitch engine
- Root note system
- Parameter locks
- Retrigger
- Fill
- Scale lock
