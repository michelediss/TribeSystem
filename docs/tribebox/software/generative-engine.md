# Generative Engine

## Goal

The generative engine creates controlled musical variation.

It should not replace manual sequencing. It should extend it.

## Core Parameters

- Root
- Scale
- Range
- Density
- Probability
- Evolution

## Relationship With Trigger Engine

The generative engine controls pitch, not rhythm.

Rhythm may still come from:

- Step triggers
- Euclidean triggers
- Trig conditions

## Use Cases

- Acid variations
- Bass movement
- Tonal utility samples
- Simple evolving patterns

## Design Principle

Generative behavior must remain controllable and musically coherent.
