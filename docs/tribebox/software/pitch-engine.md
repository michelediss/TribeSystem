# Pitch Engine

## Goal

The pitch engine manages tonal coherence inside TribeBox.

## Project Root

Each project should define a root note.

The root note is used as the tonal reference for:

- Kick selection
- Bass / rumble tuning
- Acid lines
- Generative pitch
- Tonal utility samples

## Pitch Modes

### Manual

Notes and pitch values are entered manually.

### Root Lock

A sample or synth voice is aligned to the project root.

### Scale Lock

Entered notes are constrained to a selected scale.

### Generative

Notes are generated from root, scale, range and evolution settings.

## Track Priority

Pitch engine features are most important for tracks 2, 7 and optional track 8.
