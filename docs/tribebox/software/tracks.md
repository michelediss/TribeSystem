# Tracks

## Track Layout

TribeBox uses 8 tracks.

## Default Track Roles

1. Kick
2. Bass / Rumble
3. Hat
4. Clap / Snare
5. Percussion 1
6. Percussion 2
7. Acid / Lead
8. Utility / Samples

## Output Routing

Default routing uses 4 output groups:

- Out 1: Kick
- Out 2: Bass / Rumble
- Out 3: Drums
- Out 4: Acid / Synth / FX

## Tonal Priority

Tonal management is most important for:

- Kick
- Bass / Rumble
- Acid / Lead
- Utility samples when tonal

Drum and percussion tracks may remain mostly free pitch.
