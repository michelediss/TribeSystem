# Controls Layout

This document describes the planned physical control layout for TribeBox.

## Main Control Groups

- Step buttons
- Track buttons
- Encoders
- Macro controls
- Transport buttons
- Performance buttons
- Display

## Step Buttons

Planned layout:

- 16 step buttons
- RGB feedback preferred

Uses:

- Step editing
- Pattern selection
- Parameter lock editing

## Track Buttons

Planned layout:

- 8 track buttons

Default tracks:

1. Kick
2. Bass / Rumble
3. Hat
4. Clap / Snare
5. Percussion 1
6. Percussion 2
7. Acid / Lead
8. Utility / Samples

## Encoders

Planned layout:

- 8 endless encoders
- Push function preferred

Uses:

- Track parameter editing
- Menu navigation
- Context controls

## Macro Controls

Planned layout:

- 8 macro controls

Example macro targets:

- Kick drive
- Bass density
- Drum crush
- Acid cutoff
- Acid resonance
- Delay amount
- Reverb amount
- Destroy

## Transport

Dedicated buttons:

- Play
- Stop
- Rec

## Performance Buttons

Dedicated or near-dedicated controls:

- Fill
- Retrigger
- Mute
- Solo
- Pattern
- Shift

## Display

Reference display:

- 7 inch touchscreen

The display is used for editing, feedback and configuration.
