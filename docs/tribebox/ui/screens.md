# Screens

This document describes the planned screen areas for TribeBox.

## Main Screen

Shows the current project, pattern, BPM and selected track.

## Track Screen

Shows parameters for the selected track.

Examples:

- Volume
- Pitch
- Decay
- Filter
- Drive
- FX amount

## Sequencer Screen

Shows the current pattern and step state.

Useful information:

- Active steps
- Track length
- Trigger mode
- Pitch mode
- Parameter locks

## Sample Browser

Used to load samples into drum and utility tracks.

## Project Screen

Used for project load, save and settings.

## System Screen

Used for device settings, audio settings and network status.
