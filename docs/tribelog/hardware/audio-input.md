# Audio Input

## Purpose

TribeLog records the stereo output from the mixer.

## Source

Possible mixer outputs:

- Tape out
- Control room out
- Dedicated stereo output if available

## Requirements

- Stereo line input
- Stable recording
- Low noise
- Reliable Linux support

## Recording Format

Initial target formats:

- WAV
- FLAC

## Notes

The recorder should not affect the main audio output of the live system.
