# OLED Display

## Purpose

The OLED display provides basic recording status.

## Suggested Information

- Recording state
- Recording duration
- Storage status
- GPS fix status
- Sync status

## Design Goal

The display should be small and readable.

It should not become a complex user interface.

## Physical Controls

The OLED can be paired with:

- REC button
- Status LED
- Optional stop button

## Notes

The first prototype can use a small I2C OLED display.
