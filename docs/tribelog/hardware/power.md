# Power

## Power Source

Inside the full Tribe System case, TribeLog should be powered by the internal power distribution.

## Standalone Option

A future standalone version may use:

- USB power bank
- Dedicated battery pack
- External USB-C power

## Requirements

- Stable 5V power
- Safe shutdown strategy
- Protection from sudden power loss

## Notes

If TribeLog is integrated in the flight case, it should use a dedicated protected power branch.
