# Raspberry Pi

## Reference Hardware

Possible options:

- Raspberry Pi Zero 2 W
- Raspberry Pi 4

## Responsibilities

- Audio recording
- GPS data reading
- Metadata generation
- Local file storage
- Optional network sync
- OLED status output
- Button and LED input/output

## Notes

A Pi Zero 2 W may be enough for a minimal recorder.

A Pi 4 gives more headroom for storage, networking and future features.
