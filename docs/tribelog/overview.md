# TribeLog Overview

TribeLog is the recording and archive module of Tribe System.

## Purpose

TribeLog records the stereo output of the mixer and stores each session with useful metadata.

## Main Features

- Stereo audio recording
- GPS metadata
- Date and time metadata
- Session metadata
- Local storage
- Optional cloud sync
- Minimal physical interface

## Input Source

The main audio source is a stereo line output from the mixer.

## Output Files

Planned files per session:

- audio recording
- metadata file

## Design Goal

TribeLog should work as a simple and reliable live recorder with automatic session documentation.
