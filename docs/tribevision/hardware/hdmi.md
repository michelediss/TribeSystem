# HDMI

## Outputs

TribeVision requires two displays:

- Projector output
- Local preview display

## Projector Output

The projector output is exposed through the external patch panel.

## Preview Output

The preview display remains internal to the system.

## Future Features

- Independent display layouts
- Mapping tools
- Calibration views
- Visual debugging overlays

## Notes

The final HDMI implementation may use native outputs and/or adapters depending on hardware constraints.
