# Wiring

## Internal Connections

- Raspberry Pi 5
- HDMI projector path
- HDMI preview monitor path
- Ethernet network
- Power distribution

## Network

TribeVision receives OSC messages from TribeBox through the internal network.

## Design Goals

- Modular wiring
- Easy maintenance
- Replaceable hardware
- Clear cable routing

## Notes

All critical connections should remain accessible for maintenance and replacement.
