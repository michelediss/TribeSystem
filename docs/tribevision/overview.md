# TribeVision Overview

TribeVision is the visual subsystem of Tribe System.

## Purpose

Generate live visuals that react to events generated by TribeBox.

## Core Technologies

- Raspberry Pi 5
- openFrameworks
- OSC communication
- Projection mapping

## Main Features

- Generative visuals
- Visual presets
- Multi-layer rendering
- Video playback support
- Smartphone mapping control
- Projector output
- Local monitor output

## Inputs

- OSC messages from TribeBox

## Outputs

- Projector HDMI output
- Local preview display
