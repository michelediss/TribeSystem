# Mapping

TribeVision should include a basic projection mapping layer.

## Purpose

The mapping layer adapts the rendered output to the projection surface.

## Planned Features

- Keystone correction
- Corner pinning
- Scale
- Rotation
- Crop
- Brightness
- Contrast
- Preset saving

## Workflow

1. Render the visual scene.
2. Pass the scene through the mapping layer.
3. Output the corrected image to the projector.

## Presets

Mapping settings should be saved as project or venue presets.

## Design Goal

The mapping system should be simple enough for fast setup before a live performance.
