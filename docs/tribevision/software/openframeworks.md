# openFrameworks

TribeVision uses openFrameworks as the main visual rendering framework.

## Role

openFrameworks is responsible for:

- Real-time rendering
- Generative visuals
- Video layer management
- Shader experiments
- Output composition
- Mapping layer integration

## Target Platform

- Raspberry Pi 5
- Linux
- Dual display setup

## Design Goals

- Stable real-time rendering
- Flexible creative coding workflow
- Direct access to graphics APIs
- Good support for custom visual experiments

## Notes

The initial target is 1080p output for the projector.

Performance tests will determine the practical number of layers and effects.
