# Smartphone Control

TribeVision should provide a simple smartphone control interface for setup tasks.

## Purpose

Allow projection mapping and system checks without connecting a laptop.

## Interface Type

Preferred approach:

- Local web interface
- Browser-based control
- No dedicated mobile app required

## Planned Controls

- Move mapping corners
- Adjust brightness
- Adjust contrast
- Select mapping preset
- Save mapping preset
- Check connection status

## Network

TribeVision may expose a local network or connect to the internal Tribe System network.

## Design Goals

- Easy setup
- Minimal dependencies
- Works from any modern smartphone browser
- Does not affect rendering stability during performance

## Live Use

The smartphone interface is mainly intended for setup mode.

During performance, the mapping should already be saved and stable.
