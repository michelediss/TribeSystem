# Visual Engine

The visual engine is the creative core of TribeVision.

## Purpose

Generate visuals that react to musical and performance events from TribeBox.

## Core Concepts

- Generative shapes
- Particles
- Geometry
- Trails
- Glitch effects
- Video layers
- Visual DNA presets

## Visual DNA

Each visual preset may define:

- Color palette
- Organism type
- Motion behavior
- Glitch profile
- Density
- Reaction style

## Audio Event Mapping

Possible mappings:

- Kick triggers pulse and compression
- Bass controls mass and density
- Hats control particles
- Acid controls geometry and distortion
- Delay controls trails
- Reverb controls atmosphere

## Design Principle

Visuals should feel like one evolving organism rather than unrelated clips.
