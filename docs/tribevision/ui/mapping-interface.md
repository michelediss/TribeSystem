# Mapping Interface

This document describes the planned mapping interface for TribeVision.

## Goal

Allow fast projector mapping and calibration without using a laptop.

## Control Device

The mapping interface is controlled from a smartphone connected to the local TribeVision network.

## Main Functions

- Move projection corners
- Adjust perspective
- Adjust scale
- Adjust rotation
- Adjust brightness
- Adjust contrast
- Save mapping preset
- Load mapping preset

## Typical Workflow

1. Connect smartphone to the TribeVision network.
2. Open the local control page.
3. Select the mapping surface.
4. Adjust the four corners.
5. Fine tune brightness and contrast.
6. Save the preset.
7. Switch to live mode.

## Interface Areas

### Preview Area

Shows the current mapped output.

### Corner Controls

Allows dragging the four projection corners.

### Preset Controls

Allows saving and loading mapping presets.

### Live Mode Control

Locks editing and returns TribeVision to performance mode.

## Design Principle

The mapping interface should be simple enough to use during setup in a venue or temporary performance space.
